# Shartnoma Bot

Telegram bot — shartnoma PDFini yuklang, Google Sheetga avtomatik yozadi.

## Muhit o'zgaruvchilari (Environment Variables)

| O'zgaruvchi | Qiymat |
|---|---|
| TELEGRAM_TOKEN | Bot tokeni |
| ANTHROPIC_API_KEY | Anthropic API kaliti |
| GOOGLE_SHEET_ID | Google Sheet ID |
| GOOGLE_CREDENTIALS_JSON | Service Account JSON (butun matn) |

## Ishlatish

1. PDF shartnomani botga yuboring
2. Bot avtomatik tahlil qiladi
3. Google Sheetga yozadi
